This project was developed as a hands-on practice exercise to strengthen my data visualization and business intelligence skills using Power BI.

## 🎯 Purpose of This Project
The goal was to simulate a real-world analytics scenario by working with healthcare-related data and building a dashboard that communicates key insights clearly and effectively.

## 📊 What I Practiced
- Designing clean and professional dashboards
- Creating KPI cards for high-level metrics
- Using geographic maps for global data visualization
- Building bar charts for country-level comparisons
- Improving layout, color consistency, and readability

## 🧠 Key Takeaways
- Learned how to present data in a way that is easy to understand for non-technical users
- Improved my ability to structure dashboards with a clear visual hierarchy
- Gained experience turning raw data into actionable insights

## 🛠 Tools Used
- Power BI
